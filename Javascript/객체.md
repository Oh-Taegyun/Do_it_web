```javascript

var numbers1 = ["one", "two", "three", "four"];
var numbers2 = Array("one", "two", "three", "four");
```

# Date 객체

```javascript
new Date(); // 현재 날짜로 설정할 경우
new Date("2020-02-25"); // 특정 날짜로 설정할 경우
```

자바 스크립트가 인식할 수 있는 날짜

1. YYYY-MM-DD 형식
2. YYYY-MM-DDTHH 형식
3. MM/DD/YYYY 형식
4. 이름 형식
